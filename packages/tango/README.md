# tango

`tango` is planned as a CLI-first native/tmux agent orchestrator.

The package is intended to provide:

- a `tango` CLI for starting, inspecting, messaging, and stopping agents;
- harness adapters, starting with `pi` and later Claude Code;
- reusable agent roles and shared prompt includes;
- a pi extension wrapper that exposes the CLI to pi as tools/commands.

See `../../docs/specs/tango-v1/design.md` for the initial design.
